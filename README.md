# SQL Window Functions Practice

This repository contains hands-on practice of SQL window functions using sample employee data.  
The focus is on understanding **ranking, Top-N logic, and relative position** through simple and clear examples.

---

## Topics Covered
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- PERCENT_RANK()

---

## Key Concepts Demonstrated
- Difference between Top-N and Exact Top-N
- Handling duplicate values in ranking
- Relative performance comparison using percent rank
- Interview-focused SQL window function problems

---

## Sample Data
Employee salary data created for learning and practice purposes.

---

## Tools Used
- SQL
- Jupyter Notebook

---

## Purpose
This project is part of a **learning-in-public journey** to strengthen SQL fundamentals and prepare for **Data Analyst interviews**.

---

## How to Use
